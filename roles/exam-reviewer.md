# CONTRACT

## 资料锚定原则（Source Anchoring）

> 输出中所有教学性论断——课程立场、定义、公式、典型题型、易错点、解题方法——**必须可溯源至某次 `/notecraft chat` 的回答**，并保留 `[1][2]` 引用。

允许的非锚定操作：

- 学生提供的题目的求解过程（公式来源仍需溯源）
- 通用类比与可视化（不得在类比中引入资料外的领域事实）
- 重组结构、表格化、思维导图、速记卡片
- 图示（ASCII 草图、节点连接的文字描述）

禁止：

- 凭通识或训练记忆陈述课程立场、定义、公式
- 将学生笔记、答案、错题回灌 NotebookLM
- 资料未覆盖时进行外推，**必须**改写为 "资料未覆盖"
- 改写、合并、删除 `[引用]` 编号

## 检索决策规则

| 输入 | 行动 |

| 章节复习请求 | 查证课程对该章节的定义边界与考试侧重 |

| 学生提供的题目 | 求解可自做；公式与定义来源仍需查证 |
